# Usage

## Import

Import selected `Nx{SELECTED}Loader` component into your component or module and then use with `<Nx{SELECTED}Loader>` tag.

<Tabs>
  <TabItem value="options-api" label="Options API" default>
```js title="app.vue"
<template>
  <div>
    // highlight-next-line
    <NxOvalLoader :isLoading="true" :size="40" />
  </div>
</template>

<script>
// highlight-next-line
import { NxOvalLoader } from '@ngeenx/nx-vue-svg-loaders';

export default {
  components: {
    // highlight-next-line
    NxOvalLoader,
  },

  data() {
    return {
      // highlight-next-line
      isLoading: true,
    };
  },
};
</script>

<style scoped lang="scss">
    // ...
</style>
```
  </TabItem>
  <TabItem value="composition-api" label="Composition API">
```js title="app.vue"
<template>
  <div>
    // highlight-next-line
    <NxOvalLoader :isLoading="isLoading" :size="40" />
  </div>
</template>

<script>
import { ref } from 'vue';

// highlight-next-line
import { NxOvalLoader } from '@ngeenx/nx-vue-svg-loaders';

// highlight-next-line
const isLoading = ref(true);
</script>

<style scoped lang="scss">
    // ...
</style>
```
  </TabItem>
</Tabs>

## Coloring

You can set custom color to the loader by passing `style.color` prop to the loader component or any parent element.
SVG loader will inherit the color from the parent element with `currentColor` value.

```html title="app.vue"
<template>
  <div>
    // highlight-next-line
    <NxOvalLoader :isLoading="isLoading" :size="40" :style="{ color: 'orange' }" />

    <!-- or -->

    // highlight-next-line
    <NxOvalLoader :isLoading="isLoading" :size="40" class="text-orange-500" />
  </div>
</template>
```
