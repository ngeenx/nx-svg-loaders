# Usage

Import selected `Nx{SELECTED}Loader` component into your component or module and then use with `<Nx{SELECTED}Loader>` tag.

```js title="app.svelte"
<script lang="ts">
  // highlight-next-line
  import { NxAudioLoader } from '@ngeenx/nx-svelte-svg-loaders';

  // ..

  // highlight-next-line
  let isLoading = true;
</script>

<div>
  <!-- ... -->

  // highlight-next-line
  <NxAudioLoader {isLoading} size={40} />

  <!-- ... -->
</div>

<style lang="scss">
  // ...
</style>
```
